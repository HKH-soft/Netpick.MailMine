## üöÄ About The Project

![MailMine Dashboard](ProductPics/mailmine-dash.png)

**MailMine** is a sophisticated web scraping and lead generation solution designed to streamline the process of gathering business intelligence. Whether you are a marketer looking for leads, a recruiter finding candidates, or a data analyst gathering insights, MailMine provides the tools to automate the extraction of emails, links, and contact information from the web.

### Key Features

*   **üï∑Ô∏è Advanced Web Scraping**: Powered by Jsoup and Selenium to handle complex websites.
*   **üéØ Smart Query Generation**: Automatically generate search queries tailored to specific industries and roles.
*   **üìä Interactive Dashboard**: Visualize your scraping jobs, success rates, and data collection in real-time.
*   **üîê Secure Authentication**: Enterprise-grade security with JWT tokens and role-based access control.
*   **‚ö° High Performance**: Asynchronous processing for handling multiple scraping jobs simultaneously.

<!-- GETTING STARTED -->
## üèÅ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have the following installed on your system:

*   **Java JDK 17+**
*   **Node.js 18+**
*   **Maven** (Optional, wrapper provided)

### Installation

1.  **Clone the repo**
    ```sh
    git clone https://github.com/HKH-soft/Netpick.MailMine.git
    ```

2.  **Backend Setup**
    ```sh
    cd Backend
    # Configure database in src/main/resources/application.yml
    ./mvnw spring-boot:run
    ```

3.  **Frontend Setup**
    ```sh
    cd Frontend
    npm install
    npm run dev
    ```

4.  **Access the App**
    Open your browser and navigate to `http://localhost:3000`.
